# Parsing HTTP/1.0 using state machine design

## Goals

- Check State Machine design
- Make vector myself
- Make test runner myself and add pretty output to it (at least something!)
- Check HTTP protocol (it's kinda old, but principles still interesting)

## Build

You need `gcc` and `make` in order to use Makefile.

```bash
gcc --version
make --version
```

You can install it if not installed yet.

```bash
# Ubuntu / Debian
sudo apt install gcc make

# Fedora / RHEL
sudo dnf install gcc make

# Arch
sudo pacman -S gcc make
```

Now finnally build:

```
make run
```
